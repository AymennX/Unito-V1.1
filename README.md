### Unito-V1.1
**Unito is a high-performance, Python-powered YouTube downloader designed for Windows, allowing users to save their favorite content in MP4 and MP3 formats with ease.**

---

## ‚ú® Features

Unito delivers a seamless media downloading experience with a clean Windows-optimized interface:

* **Dual-Format Extraction:** Save content as high-definition **MP4 (Video)** or crystal-clear **MP3 (Audio)**.
* **High-Resolution Support:** Download videos in 720p, 1080p, and even **4K** where available.
* **Audio Mastering:** Automatically converts video streams to high-bitrate audio files using FFmpeg.
* **Thumbnail Preservation:** Downloads and embeds the video thumbnail as the album art for MP3 files.
* **Batch Processing:** Queue multiple links to download entire playlists or series of videos in one go.
* **Windows Native UI:** Built with a modern, dark-themed interface that feels native to Windows 10/11.

## üõ†Ô∏è Commands & Usage

Unito is designed to be user-friendly. Once the application is launched, follow these simple steps:

### Using the Interface:
1.  **Paste URL:** Enter the YouTube link into the main input field.
2.  **Select Quality:** Use the dropdown to choose your preferred resolution or audio bitrate.
3.  **Choose Format:** Toggle between the `Video` and `Audio` tabs.
4.  **Download:** Click the **"Start Download!"** button.

### Example Link Formats:
* **Video:** `https://www.youtube.com/watch?v=example`
* **Shorts:** `https://www.youtube.com/shorts/example`
* **Playlists:** `https://www.youtube.com/playlist?list=example`
